# S1 룰 발화율 대역표 — 실측 전 선언 (2026-07-16)

> **원칙**: 이 표는 s5_c001 PHASE1 실측을 **돌리기 전에** 선언한다. 측정 후 숫자에 기준을
> 맞추는 fitting을 막기 위한 장치다 (PLAN.md §3 S1). 대역 밖 = 자동 불합격이 아니라
> **결함 조사 대상**이며, 조사 결과는 ① 생성기 결함 ② 룰 과탐 ③ 대역 오선언 셋 중
> 하나로 3분류해 기록한다.

## 기준선과 대역 규칙

- **사전 기준선**: 구 normal v46b 실측 (345,944행, `dev/active/phase1-2-code-rework/baseline_normal_v46b/rule_summary.csv`, 2026-06-30). 단위 = emitted_units / 행수.
- **【1차 실측 후 정정】 지표 정의 주의**: 분자(emitted_units)는 전표 단위, 분모는 행수 — 혼합 지표라 절대 %가 아니다(전표 단위 환산 시 약 3.2배). v46b와 동일 산식이므로 **변화 감지에는 유효**, 절대 수준 해석 금지. 전표 단위 정규화는 S1 재측정에서 교정.
- **기본 대역**: 사전율 × [1/3, ×3] (자릿수 감시). 사전율 0인 룰은 [0, 0.05%] (정상에서 조용해야 할 룰).
- **이번 세션 수정의 방향 예측** (s0→s5 데이터 변경): 예측과 다르면 그 자체가 조사 대상.
- **전역 과탐 기준**: 단일 룰이 정상 모집단의 **8% 초과**를 플래그하면 사전율과 무관하게 과탐 조사 대상 (정상 데이터에서 "감사인이 봐야 할 항목"이 될 수 없는 규모).

## 판정 규칙 (S1-2에서 적용)

1. 실측이 대역 안 → PASS 기록만.
2. 대역 밖 → 룰별로 ① 생성기 결함 ② 룰 과탐 ③ 대역 오선언 3분류. ③으로 판정하려면 도메인 근거 필수(숫자 맞추기 금지).
3. **L1-05 = 선판정 결함**(D3 미구현, s5 전수 0건 확인) — 실측 확인 후 생성기 수정 루프로.
4. 전역: HIGH tier 문서 비율과 룰당 8% 상한은 별도 집계로 보고.
