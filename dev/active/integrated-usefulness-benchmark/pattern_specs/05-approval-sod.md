# pattern_spec 5 — 승인 SoD (자기승인·직무분리 위반)

모집단: in-scope 패턴족=승인SoD (multi-label 24). Phase 2, 상태 중량.
대표 case: `2018-2019-17`(대표가 구매팀 폐지→차명회사 단독 통제), `2010-가-Aa`(자금팀 직원이 160여 회 무단이체), `A5-나-K`(자금팀장이 입출금·승인·잔액조회 단독 수행), `2012-바-A`(이사회결의·계약서 없이 수시 송금), `2011-나-나`(재무담당임원이 회사자금 유용).

## 기전 (control flow)
전표의 생성·승인·기록이 한 사람(또는 공모 2인)에게 집중되어 통제가 무력화된다. 전형: (a) **자기승인**(지출 생성자가 곧 승인자), (b) **직무분리 부재**(구매팀 폐지, 자금 입출금·거래승인·잔액조회를 1인이 수행), (c) **이사회 결의·계약서 등 상위 승인근거 부재**, (d) 상급자 단독 지시로 하급자가 실행. 단독 family로도, 횡령(spec 4)·가공(spec 1)의 통제축으로도 발현.

## A — 기전 고정값 (5벌 동일)
| 컬럼                | 값                                       | 근거      |
| ------------------- | ---------------------------------------- | --------- |
| approved_by         | = created_by (자기승인) 또는 상급자 단독 | 직무 결합 |
| user_persona        | 자금팀장·재무임원·대표이사               | 사례 공통 |
| supporting_doc_type | 위·변조 증빙 또는 누락                   | A5-나-K   |
| sod_violation       | true (truth 플래그)                      | 의도 위반 |

## C — 현실 범위 (5벌 재추출)
| 컬럼          | 범위                               |
| ------------- | ---------------------------------- |
| debit_amount  | 결합된 횡령·가공 금액에 종속       |
| posting_date  | 상시(무단이체는 반복·다회)         |
| approval_date | posting과 동시·즉시(급속 자기승인) |

## D — 상태 대상 (강함)
- **이사회 결의·계약서 부재**: 상위 승인 문서체인을 참조해 *없음*을 구성(존재해야 할 근거의 부재).
- **반복 무단이체**: 동일 주체의 다회 전표가 누적 — 주체·계좌 상태 일관.
- 횡령·가공과 동반 시 그 family의 상태 대상 공유.

## spec 선언위반 (= truth)
1. 전표 **생성자와 승인자가 동일**(직무분리 위반).
2. **이사회 결의·계약서 등 상위 승인근거가 부재**한데 자금이 집행된다.
3. 증빙이 **위·변조되거나 장부에서 누락**된다.
4. 한 사람이 **자금 입출금·거래승인·잔액조회를 모두 수행**한다.
> 그 외 정합은 지킨다. rule-blind: 위반은 승인주체 동일성·상위근거 부재·직무결합의 사실이며 탐지 임계가 아니다.
