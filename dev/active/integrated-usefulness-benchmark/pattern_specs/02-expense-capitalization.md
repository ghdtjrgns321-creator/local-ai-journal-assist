# pattern_spec 2 — 비용자산화 (expense capitalization)

모집단: in-scope 패턴족=비용자산화 (multi-label 18). Phase 1, 상태 경량.
대표 case: `2011-라-나`(노무비·소모품비를 시운전비·기계장치 취득부대비용으로), `FSS2206-08`(외주가공비를 유형자산으로, 계약서·세금계산서 위조), `FSS2311-10-01`(외주가공비를 유형자산, 무증빙 손익조정 분개), `A-12유형-가`(이자비용 전액 자본화), `2018-2019-12`(연구개발비를 개발비 자산화).

## 기전 (money/document flow)
당기비용으로 처리해야 할 지출(외주가공비·노무비·소모품비·연구개발비·이자·환율손실)을 자산 계정(유형자산·무형자산·건설중인자산·개발비)으로 계상한다. 비용이 자산으로 이연되어 당기 이익·자기자본이 과대된다. 흔히 관리종목·상장폐지 회피 목적. 자산화 분개는 결산 시 수기로 입력되고, 증빙(계약서·품의서·세금계산서)은 위조되거나 아예 없다.

## A — 기전 고정값 (5벌 동일)
| 컬럼                | 값                                                                    | 근거                                                                   |
| ------------------- | --------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| gl_account          | 차) 자산(유형자산·무형자산·건설중인자산·개발비) / 대) 비용계정·현금성 | 비용→자산이 정체성                                                     |
| document_type       | 수기 결산조정·대체전표(SA)                                            | 결산 손익조정 분개                                                     |
| source              | 결산 수기조정 subtype은 Manual/adjustment, 그 외는 base source 유지   | 전건 Manual 금지 — 정상 결산조정에도 Manual 존재, fraud 단독 tell 방지 |
| user_persona        | 회계팀장·재무                                                         | FSS2311-10-01·FSS2206-08                                               |
| supporting_doc_type | 위조 계약서·품의서·세금계산서 또는 **없음**                           | 무증빙 손익조정                                                        |

## C — 현실 범위 (5벌 재추출)
| 컬럼         | 범위                                                      |
| ------------ | --------------------------------------------------------- |
| debit_amount | 자본화한 비용 규모(중요성 하한 이상, 이익 전환 목표 금액) |
| posting_date | 결산기 집중 또는 상시 누적                                |

## D — 상태 대상 (Phase1 경량)
대부분 **새 자산을 생성**하므로 상태의존 낮다. 예외: "건설중인자산→기계장치 대체"(A-3유형-나)는 기존 CIP 잔액을 참조해 대체 — 그 경우 실재 CIP 객체 참조.

## spec 선언위반 (= truth)
1. 자산 계정에 계상됐으나 지출의 **성격이 당기비용**(소모·노무·외주가공·연구·이자)이다.
2. 자산화 **요건(미래경제효익·기술적 실현가능성)을 충족하지 못한다**.
3. (가공자산형) 자산화에 대응하는 **실물 자산이 없다**(취득사실 없는 기계장치).
4. 자산으로 옮겨 **감가상각·비용 인식을 지연**시킨다.
> 그 외 정합(차대균형·계정형식)은 지킨다. rule-blind: 위반은 지출 성격·자산 실재·요건 충족의 사실이다.
