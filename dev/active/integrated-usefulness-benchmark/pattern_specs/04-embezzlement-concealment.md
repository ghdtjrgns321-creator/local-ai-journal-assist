# pattern_spec 4 — 횡령은폐 / 중복자금유출 (embezzlement & concealment)

모집단: in-scope 패턴족=횡령은폐 (multi-label 88). Phase 2, 상태 중량.
대표 case: `2018-2019-16`(대표이사 선급금 인출→paper company 송금→재입금 위장), `2018-2019-17`(차명회사 G사 선급금 30% 선지급·횡령), `2010-다-C`(자금담당이 AR 무단 할인 유용→다른 AR로 돌려막기), `2010-다-P`(임원 보수를 자문료 선급금 허위계상), `C-1유형-라`(대표 단기대여 후 회수, 가수금 유사).

## 기전 (money/control flow)
회사 자금을 빼돌리고 그 흔적을 가린다. 전형: (a) 대표/자금담당이 **선급금·대여금 명목으로 자금 인출**, (b) **paper company·차명회사를 경유**해 재입금해 정상 거래처럼 위장, (c) 가수금·미수금으로 **장기 미해소** 방치, (d) 횡령한 매출채권의 만기에 **다른 매출채권을 할인해 돌려막기**. 대응 실물·계약 실질이 없다.

## A — 기전 고정값 (5벌 동일)
| 컬럼                | 값                                    | 근거            |
| ------------------- | ------------------------------------- | --------------- |
| gl_account          | 선급금·대여금·가수금(suspense)·미수금 | 자금 은닉 계정  |
| user_persona        | 대표이사·자금담당(차장/팀장)          | 사례 공통       |
| approved_by         | = created_by(자기승인) 또는 무승인    | 통제 우회       |
| counterparty_type   | paper company·차명회사·특수관계자     | 2018-2019-16/17 |
| supporting_doc_type | 없음 또는 위조 계약서                 | 지급근거 부재   |
| is_suspense_account | true (가수금형)                       | 장기체류        |

## C — 현실 범위 (5벌 재추출)
| 컬럼         | 범위                                        |
| ------------ | ------------------------------------------- |
| debit_amount | 인출 규모(이상고액 빈번), 분할 시 한도 직하 |
| posting_date | 상시 + 은폐(결산 직전 회수 위장)            |

## D — 상태 대상 (강함 — Phase 2 선행 오라클 필요)
- **돌려막기**: 기존 *열린 매출채권*을 참조해 "할인/회수 위장"(없는 AR 갚기 금지).
- **순환 재입금**: 유출 전표 ↔ 재입금 전표 **쌍**이 동일 자금으로 연결.
- **가수금 장기체류**: 미결제(clearing 미완) 상태를 기간에 걸쳐 유지.
- **역분개 위장**: 기말 회수처럼 처리 후 익기초 역분개 — *실재 원전표* 참조.

## spec 선언위반 (= truth)
1. 선급금·대여금인데 대응하는 **PO·GR·계약 실질이 없다**.
2. 자금 유출과 재입금이 **동일 자금의 순환**(외부 실수요 없음).
3. 가수금이 정당한 사유 없이 **장기 미해소**된다.
4. 매출채권 "회수" 기록이 **실제 현금 유입과 불일치**(돌려막기).
5. 지출 생성자와 승인자가 동일(→ spec 5와 결합).
> spec 4 위반만 깨고 차대균형·금액형식·계정짝 정합은 지킨다. rule-blind: 위반은 자금흐름 폐쇄성·증빙 실질·clearing 상태의 사실이다.
